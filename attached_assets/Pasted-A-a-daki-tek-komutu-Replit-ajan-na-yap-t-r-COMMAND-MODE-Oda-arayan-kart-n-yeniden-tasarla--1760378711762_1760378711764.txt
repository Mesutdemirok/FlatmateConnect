Hedef

“Oda Arayanlar” bölümünde, sol üstte görünecek modern bir oda arayan kartı. Üstte yuvarlak fotoğraf, altında isim; kart kenarlarında bütçe, tercih edilen bölge, meslek ve yaş için rozet/pill alanları.

⸻

Yapılacaklar

1) Yeni bileşen
	•	Dosya: client/src/components/ui/SeekerCard.tsx
	•	Props:

interface SeekerCardProps {
  seeker: {
    id: string
    fullName: string // ad + soyad (soyad ilk harf de kabul)
    avatarUrl?: string
    age?: number
    occupation?: string // öğrenci / çalışan vb.
    preferredLocations?: string[] // ilçe/mahalle listesi (ilk 2 göster)
    budgetMonthly?: number // ₺
    verified?: boolean
  }
  onClick?: () => void
}


	•	Yapı:
	•	Dış çerçeve: rounded-2xl bg-white shadow-sm ring-1 ring-black/5 hover:shadow-lg transition
	•	Üst kısım (foto):
	•	Orta hizada yuvarlak görsel: w-28 h-28 md:w-32 md:h-32 rounded-full object-cover ring-4 ring-white shadow
(mevcut görselden bir tık daha büyük, ama kartı boğmayacak ölçü)
	•	Avatar yoksa baş harflerle placeholder.
	•	Doğrulama varsa sağ altına küçük onay rozeti: absolute -bottom-1 -right-1 rounded-full bg-rose-500 text-white p-1 shadow
	•	İsim: fotoğraf altı merkezde, text-xl md:text-2xl font-bold text-slate-900
	•	Kenar rozetleri (4 bilgi): kartın iç kenarlarına sabitle:
	•	Sol-üst: Tercih edilen bölge (ilk 1–2 yer, “…“ ile kısalt)
absolute top-3 left-3 rounded-full bg-indigo-50 text-indigo-700 text-xs font-medium px-2.5 py-1
	•	Sağ-üst: Bütçe: ₺xx.xxx/ay
absolute top-3 right-3 rounded-full bg-emerald-50 text-emerald-700 text-xs font-semibold px-2.5 py-1
	•	Sol-alt: Meslek (Öğrenci/Çalışan vs.)
absolute bottom-3 left-3 rounded-full bg-slate-100 text-slate-700 text-xs font-medium px-2.5 py-1
	•	Sağ-alt: Yaş: xx yaş
absolute bottom-3 right-3 rounded-full bg-fuchsia-50 text-fuchsia-700 text-xs font-medium px-2.5 py-1
	•	Kart yüksekliği: görselin etrafında yeterli boşluk için p-5 md:p-6 + relative ebeveyn.
	•	Erişilebilirlik: avatar alt={fullName}, kart role="button" ve aria-label="Profili aç: {fullName}".

	•	Mobilde avatar w-28 h-28, desktop’ta w-32 h-32.
