╚══════════════════════════════════════════════════════════════════╝

PROBLEM: www.odanent.com.tr serves static files only (no backend/API)
SOLUTION: Point domain to Autoscale deployment (runs Express server)

┌──────────────────────────────────────────────────────────────────┐
│  STEP 1: Open Replit Deployments                                 │
└──────────────────────────────────────────────────────────────────┘

1. Click "Deploy" button (top right in Replit)
2. Look at your existing deployments

You'll see something like:

   ┌─────────────────────────────────────────┐
   │ 📦 Static Deployment                    │
   │ Type: Static                            │  ← THIS IS THE PROBLEM!
   │ Domain: www.odanent.com.tr             │
   │ Status: Active                          │
   └─────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────────┐
│  STEP 2: Delete Static Deployment & Create Autoscale            │
└──────────────────────────────────────────────────────────────────┘

A) DELETE the static deployment:
   - Click "..." or settings icon
   - Select "Delete deployment"
   - Confirm deletion

B) CREATE new Autoscale deployment:
   - Click "+ New Deployment" or "Create Deployment"
   - SELECT: "Autoscale" (NOT "Static"!)
   
   ┌─────────────────────────────────────────┐
   │ Choose Deployment Type:                 │
   │                                         │
   │ ✅ Autoscale  ← CHOOSE THIS!           │
   │    Runs Node.js server                  │
   │    Handles /api routes                  │
   │                                         │
   │ ❌ Static                               │
   │    Only serves HTML/CSS/JS              │
   └─────────────────────────────────────────┘

   - Build: npm run build (auto-filled)
   - Run: npm run start (auto-filled)
   - Click "Deploy"

┌──────────────────────────────────────────────────────────────────┐
│  STEP 3: Connect Custom Domain                                   │
└──────────────────────────────────────────────────────────────────┘

After deployment succeeds:
1. Click on the new Autoscale deployment
2. Find "Domains" or "Custom Domains" section
3. Click "Add custom domain"
4. Enter: www.odanent.com.tr
5. Save

┌──────────────────────────────────────────────────────────────────┐
│  VERIFY IT WORKS (Wait 2-5 minutes)                              │
└──────────────────────────────────────────────────────────────────┘

Test these URLs:

✅ https://www.odanent.com.tr/api/health
   Should return: {"status":"ok"}

✅ https://www.odanent.com.tr/api/listings  
   Should return: JSON array with listings

✅ https://www.odanent.com.tr
   Should show full website with data

┌──────────────────────────────────────────────────────────────────┐
│  WHAT'S THE DIFFERENCE?                                          │
└──────────────────────────────────────────────────────────────────┘

STATIC (Current - Wrong):
www.odanent.com.tr → Only HTML/CSS/JS files
                  → NO /api routes ❌
                  → NO database ❌
                  → NO Express server ❌

AUTOSCALE (Need This - Correct):
www.odanent.com.tr → Express Node.js Server ✅
                  → Serves /api routes ✅
                  → Database access ✅
                  → Frontend + Backend together ✅
